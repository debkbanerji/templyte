New User Tutorial

Create a Template

  1. Navigate to the "Create Template" page from the navigation bar at the top of each page
  2. Provide a name and description of your template
  3. To create your template, write code as normal but make sure to put curly braces around any variable names you want to be customized. Make sure to zip all relevant files
  4. Select the zip file of the template
  5. Provide Names and Descriptions of Template Variables. Template Variables are the customizable parts of the template
  6. Provide Tags, used for searching
  7. Provide File Endings. The file endings describe which files have variable names in them and therefore which files the back end looks at when it's populating the template during download
  8. Submit template

View Templates

  1. Navigate to the "My Templates" page from the navigation bar at the top of each page
  2. Your existing templates will be displayed, if there are none you have not created any

Download A Template

  1. Navigate to the "My Templates" page from the navigation bar at the top of each page
  2. Your existing templates will be displayed, if there are none you have not created any
  3. If you wish to download one of your templates, click on download button
  4. Fill in your customizations and download

Delete A Template

  1. Navigate to the "My Templates" page from the navigation bar at the top of each page
  2. Your existing templates will be displayed, if there are none you have not created any
  3. If you wish to delete one of your templates, click on delete button

Rate A Template

  1. Navigate to the "My Templates" page from the navigation bar at the top of each page
  2. Your existing templates will be displayed, if there are none you have not created any
  3. If you wish to rate one of your templates, click on the template
  4. Click the stars to rate
