New User Tutorial

Create a Template

  1. Write code as normal but make sure to put {{ around any variables you want to be customized
  2. Make sure to zip all relevant files

More details on variable marking

All variables are HTML-escaped by default. If you want to render unescaped HTML, use the triple mustache {{{ to enclose the variable. You can also use & to unescape a variable

If you want to use {{ in your code and not have it represent a variable, you will have to change the delimeter to something else and then change it back after that block of code. You can do that by assigning it to a new symbol with an =

Always make sure to set the delimeter back to the original afterwards

For more information about variables visit https://mustache.github.io/mustache.5.html

Upload a Template

  1. Navigate to the "Create Template" page from the navigation bar at the top of each page
  2. Provide a name and description of your template
  3. Select the zip file of the template
  4. Provide Names and Descriptions of Template Variables. Template Variables are the customizable parts of the template, and are indicated by {{ around them
  5. Provide Tags. They are used for searching
  6. Provide File Endings. The file endings describe which files have variable names in them and therefore which files the back end looks at when it's populating the template during download
  7. Submit template

View Templates

  1. Navigate to the "My Templates" page from the navigation bar at the top of each page
  2. Your existing templates will be displayed, if there are none you have not created any

Download A Template

  1. Navigate to the "My Templates" page from the navigation bar at the top of each page
  2. Your existing templates will be displayed, if there are none you have not created any
  3. If you wish to download one of your templates, click on download button
  4. Fill in your customizations and download

Delete A Template

  1. Navigate to the "My Templates" page from the navigation bar at the top of each page
  2. Your existing templates will be displayed, if there are none you have not created any
  3. If you wish to delete one of your templates, click on delete button

Rate A Template

  1. Navigate to the "My Templates" page from the navigation bar at the top of each page
  2. Your existing templates will be displayed, if there are none you have not created any
  3. If you wish to rate one of your templates, click on the template
  4. Click the stars to rate
